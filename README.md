# landing-biteradigital-pages-dev
Esto es un proyecto Next.js: TypeScript, App Router, Tailwind, componentes reutilizables. Cada sección es su propio archivo .tsx. Cuando querés cambiar el copy de los CTAs, tocás CTAFinal.tsx. Cuando querés agregar una sección de casos de uso, creás UseCases.tsx y lo importás en page.tsx. Compila para Cloudflare Pages.
